# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in **goa--key-api-citizen0service**, please report it responsibly.

- Email: security@govalta.ca
- Do NOT create a public GitHub issue for security vulnerabilities

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |

## Security Measures

- All dependencies monitored via Dependabot
- Automated security fixes enabled
- Branch protection requires code review
- Security headers enforced (Helmet)
- Rate limiting enabled
- Input validation on all endpoints
- Generated by Pronghorn 🦌 with enterprise security defaults
